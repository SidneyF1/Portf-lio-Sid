Minha Corrida: Projeto p5.js

O projeto é uma página web criada com p5.js, destinada a simular uma corrida entre dois carros representados por emojis. O objetivo é criar uma experiência visual simples e interativa, onde cada carro se move verticalmente em direção à linha de chegada. A tela possui um fundo azul que remete ao céu, e os carros são posicionados em colunas distintas para a disputa. Quando um dos carros atinge a linha de chegada, uma mensagem indica qual deles venceu a corrida.

A estrutura do projeto é baseada em JavaScript com a biblioteca p5.js, que define a lógica da corrida e a renderização dos elementos visuais na tela. O desenho e movimentação dos carros, assim como o fundo e o texto, são gerados dinamicamente no canvas criado pela biblioteca. As cores, posições e interações são controladas diretamente no código, proporcionando uma experiência visual simples e fluida.

Recomendador de Filmes: Projeto p5.js

O projeto consiste em uma aplicação web interativa, desenvolvida com a biblioteca p5.js (que utiliza HTML, CSS e JavaScript), que funciona como um recomendador de filmes personalizado. O objetivo é oferecer aos usuários uma experiência divertida e dinâmica, na qual eles recebem sugestões de filmes com base na idade e nas preferências de gênero, como fantasia e aventura.a.

O HTML, gerado por meio das funções da p5.js, organiza os elementos da interface, como campos de entrada para idade, caixas de seleção para os gêneros favoritos e a área onde a recomendação de filme é exibida. O CSS, embora não diretamente configurado no código, pode ser aplicado externamente para estilizar a interface, garantindo um design limpo, atrativo e responsivo, que melhora a experiência do usuário. O JavaScript, com o suporte da p5.js, é responsável pela lógica da aplicação, controlando o comportamento dos botões, leitura dos dados inseridos e exibição dinâmica das recomendações.

O funcionamento da aplicação é simples e intuitivo: o usuário informa sua idade e seleciona se gosta dos gêneros fantasia e aventura. Com base nessas informações, a função geraRecomendacao() processa a lógica e retorna um filme adequado ao perfil do usuário. A recomendação é então exibida no centro da tela de forma clara e destacada.

A construção desse projeto demonstra como HTML, CSS, JavaScript e bibliotecas como p5.js podem ser integrados para criar experiências digitais interativas, que não apenas divertem, mas também exploram conceitos de lógica, programação e personalização de conteúdo.

Corrida aquática: Projeto p5.js

Este projeto, desenvolvido em p5.js, cria uma experiência interativa de corrida aquática onde quatro jogadores competem para chegar à linha de chegada. Cada jogador controla um personagem representado por um emoji, como um barco (🚣), um surfista (🏄), um nadador (🏊), e uma lancha (🛥). A movimentação dos personagens é controlada pelas teclas "A", "S", "D" e "F", respectivamente para cada jogador. A cada vez que uma tecla é pressionada, o personagem avança uma distância aleatória, tornando o jogo dinâmico e imprevisível.

O cenário é ambientado em uma pista aquática com uma linha de chegada visível, onde a disputa é marcada por uma série de retângulos que simulam ondas ou marcações na água. Quando um jogador ultrapassa a linha de chegada, uma mensagem indicando o vencedor é exibida na tela e o jogo é encerrado, garantindo que o vencedor seja claramente identificado. A mudança de fundo, dependendo se a janela do navegador está focada ou não, adiciona um toque visual ao ambiente de jogo, tornando a experiência ainda mais imersiva.

Este jogo simples mas envolvente combina velocidade, estratégia e um pouco de sorte, promovendo diversão em uma corrida animada entre amigos ou jogadores.
